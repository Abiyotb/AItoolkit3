How to publish online

  1. Upload the folder to any static host (e.g., an internal web server, SharePoint static hosting, GitHub Pages, or an S3 bucket).
  2. Share the index.html link as the entry point.
  3. Optional: configure single sign-on at the hosting layer if you need access control.
Data privacy note
Forms save drafts to localStorage in the user’s browser. No data is sent anywhere by this site. If you add a backend later, ensure you apply your organization’s privacy and security requirements.
